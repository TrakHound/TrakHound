# Timestamp Entity

Timestamp Entities are used to store a single Timestamp.

## UUID
A UUID property follows the below format:
```
SHA256({ObjectUuid}:timestamp)
```

## JSON Format
```json
[
  {ObjectUuid},
  {Value},
  {SourceUuid},
  {Created}
]
```
